# FreeOCR – Free & Open Source OCR Tool

FreeOCR là một công cụ OCR **miễn phí, mã nguồn mở**, chạy **offline**, giúp:
- OCR PDF / ảnh
- Tạo Searchable PDF (giữ nguyên layout)
- Không quảng cáo
- Không thu thập dữ liệu
- Không giới hạn file

## Vì sao dự án này tồn tại?
Hiện nay nhiều website OCR:
- Bắt trả phí
- Giới hạn số trang
- Upload tài liệu nhạy cảm lên server

FreeOCR sinh ra để:
- Ai cũng dùng được
- Chạy trên máy cá nhân
- Minh bạch mã nguồn

---

## Tính năng
- OCR PDF scan
- Giữ nguyên bố cục
- Hỗ trợ nhiều ngôn ngữ (English, Vietnamese…)
- CLI đơn giản
- 100% offline
